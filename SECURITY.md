# Security Policy

## Scope

This repository is an unofficial research archive. It is not an official Anthropic product or support channel.

Use this policy for:

- Sensitive repository-specific issues
- Accidental disclosure of secrets or personal data in this repository
- Harmful or misleading repository scripts, docs, or published materials

Do not use this repository to report:

- Security issues in Anthropic-hosted products or services
- Account issues, billing issues, or official product support requests
- General rights or takedown requests that belong in [NOTICE.md](NOTICE.md)

## How to Report

- Sensitive repository-specific issues: email `arain.shjia@gmail.com`
- Non-sensitive repository bugs: open a GitHub issue using the appropriate template
- Rights or takedown requests: follow [NOTICE.md](NOTICE.md)

## What to Include

- A short summary of the issue
- Affected file paths, docs, scripts, or workflows
- Steps to reproduce, if applicable
- Impact and any suggested safe remediation

## Response Expectations

- Best-effort review by the maintainer
- No guaranteed SLA
- If the report is actually about an official Anthropic product, you may be redirected to Anthropic's official reporting path

If you accidentally exposed a secret, rotate it immediately before or while reporting the issue.

# 安全说明

## 适用范围

本仓库是非官方研究档案，并不是 Anthropic 官方产品或支持渠道。

本说明适用于：

- 仓库自身的敏感安全问题
- 本仓库中意外暴露的密钥、令牌或个人数据
- 仓库脚本、文档或发布材料中可能带来风险的内容

以下问题不要在这里提交：

- Anthropic 官方产品或服务的安全漏洞
- 账号、计费或官方产品支持问题
- 更适合按照 [NOTICE.md](NOTICE.md) 处理的权利或下架请求

## 如何报告

- 仓库自身的敏感安全问题：发送邮件至 `arain.shjia@gmail.com`
- 非敏感仓库问题：通过 GitHub issue 模板提交
- 权利或下架请求：请参阅 [NOTICE.md](NOTICE.md)

## 建议提供的信息

- 问题简述
- 受影响的文件路径、文档、脚本或工作流
- 如可复现，请提供复现步骤
- 影响范围以及建议的安全修复方式

## 响应预期

- 由维护者尽力处理
- 不提供固定 SLA
- 如果问题实际上属于 Anthropic 官方产品，可能会建议你改走 Anthropic 官方渠道

如果你意外暴露了密钥，请在报告之前或同时立即完成轮换。
